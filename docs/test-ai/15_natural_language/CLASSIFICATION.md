- **Từ vựng không khớp**: AI quá cứng nhắc khi gặp từ đời thường ("món" → "sản phẩm", "thùng sữa" → "sản phẩm", "đồ uống" → "loại sản phẩm")
- **Không tìm thấy dữ liệu**: Nhiều câu trả về "không tìm thấy" dù có thể do điều kiện lọc sai
- **Chức năng chưa hỗ trợ**: AI từ chối nhiều tính năng (hết hạn, inactive, giảm giá, nợ NCC) — có thể do catalog chưa mapping đúng
- **Multi-turn yếu**: Các câu follow-up (N51, N56, N112) thường không tính được do mất context

### Loại 3 — Lỗi kỹ thuật
- **max_sql_attempts**: Hết 3 lần thử SQL → cần tăng limit hoặc optimize query
- **Timeout**: N33 bị timeout sau 124s → cần optimize query hoặc tăng timeout
- **Kết nối DB**: Một số câu báo lỗi kết nối → cần kiểm tra connection pool
